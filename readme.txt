      Image Text Extraction and Summation App
This app uses EasyOCR and Streamlit to extract text from images and calculate the sum of numbers found in the image.

Features:
  Upload an image (PNG, JPG, JPEG).
  Extract text from the image using EasyOCR.
  Extract all numbers from the text and calculate their sum.
  View the extracted text and the sum of numbers in the image.

Requirements:
  Python 3.x
  streamlit
  easyocr
  pillow
  numpy
